A full-stack web application to browse movies, view details, and submit reviews. Built to showcase full-stack development skills including React, Tailwind CSS, Node.js, Express, API integration, and database management.

🎯 Purpose

I’ve always loved watching movies and sharing my thoughts through reviews. This project combines my passion for movies with my full-stack development skills. Users can browse movies, see detailed information, and write reviews that are stored in a database for future reference.

🛠 Tech Stack

Frontend:

React (component-based architecture)

Tailwind CSS (responsive, utility-first styling)

React Router (dynamic routing)

Axios / fetch (API requests)

Backend:

Node.js

Express.js (REST API)

Routes for review creation & retrieval

Input validation & error handling

Database:

Persistent storage for user reviews (MongoDB / your choice of DB)

CRUD operations for reviews

Associating reviews with movies

External API:

TMDB API for real-time movie data

Search, top-rated, and movie detail endpoints

⚙️ Features

Browse movies (Home & Top Rated sections)

Search for movies dynamically

View detailed information for each movie

Write reviews (Name, Rating /10, Review text)

Store and retrieve reviews from the database

Navigate from reviews back to their corresponding movie

🎯 Learning Outcomes

Structuring a full-stack application with a separate frontend & backend

Designing RESTful APIs

Integrating third-party APIs (TMDB)

Managing database CRUD operations

Handling asynchronous data flow and state in React

Building a responsive UI with Tailwind CSS
